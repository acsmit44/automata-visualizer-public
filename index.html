Automaton Visualizer

A helper web applet for visualizing Finite Automaton.

Create Automaton

Please input a regular expression to use to generate the finite automaton. The reserved characters for the regular expression are $, +, *, (, and ). $ is the empty string (or epsilon), + is the OR operator, * is the Kleene star, and parentheses group expressions. Only alphanumeric characters and the meta characters mentioned about will be accepted as valid input. All other characters will cause an error.

Run Visualizer on Input

Please input a string to consume with the finite automaton. The buttons are mostly self-explanatory, but I will explain each purpose just to be sure:

  • Start - begins the automaton but does not consume input.
  • Progress - consume a single character and progress the automaton one state forward.
  • Undo - take one step back in the vizualizer.
  • Run to end - run the automaton automatically, consuming characters sequentially until the automaton terminates.
  • Reset - stop the automaton and fully reset so that no characters have been consumed.
